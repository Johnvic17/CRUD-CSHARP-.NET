# 🗄️ Desktop CRUD Application

A robust, full-featured desktop application developed in **C#** using **Windows Forms** and integrated with a **MySQL** database. This project implements complete CRUD (Create, Read, Update, Delete) operations, showcasing practical knowledge in desktop software architecture, local server configuration, and relational database management.

---

## 🚀 Overview

This application provides a seamless, graphical user interface (GUI) designed to interact directly with a MySQL server. By configuring a local environment via **XAMPP**, the project demonstrates a secure, efficient connection workflow, focusing on industry best practices like clean code organization, proper connection pooling, and secure SQL command execution.

---

## 🛠️ Core Features & Technical Skills

- **Desktop Development:** Built using C# and the .NET framework with a Windows Forms graphical interface.
- **Database Integration:** Full integration with MySQL database using native MySQL connector libraries.
- **Complete CRUD Lifecycle:** Robust implementation for creating, reading, updating, and deleting data records.
- **Environment Management:** Local server setup, database hosting, and service management using XAMPP.
- **Software Architecture Best Practices:** 
  - Structured and organized codebase.
  - Efficient database connection management (opening/closing states safely).
  - Prevention of common vulnerabilities through secure SQL command practices.

---

## 🧠 Key Takeaways

Developing this project consolidated essential backend and frontend concepts for desktop engineering, including:
- **Data Persistence:** Structuring relational tables and executing optimized queries.
- **State Management:** Keeping the user interface in sync with real-time database updates.
- **Full-Stack Desktop Flow:** Connecting user input directly to a structured local server securely.

## 💻 Getting Started

### Prerequisites
- [.NET SDK / Visual Studio](https://visualstudio.microsoft.com/) (with Windows Forms desktop development workload)
- [XAMPP](https://www.apachefriends.org/) (with Apache and MySQL modules enabled)
- MySQL Connector/NET (usually installed via NuGet Package Manager in Visual Studio)

### Local Setup
1. **Database Configuration:**
   - Open the XAMPP Control Panel and start **Apache** and **MySQL**.
   - Go to `localhost/phpmyadmin` in your browser and create your database schema.
